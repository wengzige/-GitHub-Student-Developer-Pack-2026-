## 🤔 为什么你的申请总是不通过？

很多同学用国内edu邮箱申请秒拒，或者一直卡在位置验证，原因通常是：
1.  **IP与位置不符**：开了代理，或者IP定位和学校不在一个城市。
2.  **资料不一致**：GitHub Profile 里的名字和 billing 里的名字、学生证上的名字对应不上。
3.  **缺少英文辅助**：GitHub 的审核系统（或人工审核员）看不懂纯中文的学生证。

本教程亲测有效，核心思路是：**“全英文资料 + 真实edu邮箱 + 远程学习(Distance Learning)声明”**。

## 🛠️ 准备工作

* **本人的 edu.cn 邮箱注册的满3天的github账号**（强烈建议用学校分配的，购买的邮箱极易封号）。
* **电脑摄像头**（申请过程中需要实时拍照）。
* **关闭所有代理/VPN**（非常重要！开全局也没用，必须关掉）。

---

## 📝 第一步：完善 Public Profile

进入 GitHub 的 **Settings** -> **Public profile**。
这一步是为了建立“真实账号”的指纹信息。

* **Name**: 填写真实姓名的**小写拼音**（例如：`wangxiaoming`），不要用网名 。
* **Public email**: 选择你的 `xxxx@xxx.edu.cn` 邮箱 。
* **Bio (个人简介)**: 建议用英文写一段简单的介绍，包含学校和专业。
    * *参考模板*: `A junior in [你的学校英文名] University, majoring in [你的专业英文名], enjoys programming.` 
* **URL**: 填写你学校的官网地址（例如 `https://www.tsinghua.edu.cn`）。
* **Company**: 填写 **学校英文全称** 。
* **Location**: 填写 **学校所在城市的英文名**（例如 `Beijing, China`）。
* **Time zone**: 务必选择 `(GMT+08:00) Beijing` 。

## ⚙️ 第二步：修改用户名

进入 **Settings** -> **Account**。

* **Change username**: 如果是新号，建议将用户名改为**真实姓名的拼音**（如 `wangxiaoming01`）。这能大幅提高审核通过率，让名字、邮箱、Billing信息三位一体 。
* *注意：如果是老号且有很多仓库依赖，改名会导致旧链接失效，请慎重操作。*

## 💳 第三步：设置 Billing Information

GitHub 会校验你的账单地址和学校地址是否大概一致。

进入 **Settings** -> **Billing and licensing** -> **Payment information** -> **Billing information**。

* **First name / Last name**: 填写和第一步完全一致的**拼音** 。
* **Address**: 填写学校的英文地址（可以用翻译软件翻一下学校地址）。
* **City / Country / State**: 全部按**学校所在地**如实填写 。

## 🎓 第四步：发起认证申请

1.  进入 **Settings** -> **Billing and licensing** -> **Education benefits**。
2.  点击 **"Verify my student status"** (或 Start an application)。
3.  **身份选择**：
    * Select your role: **Student** 。
    * Select this school: 选中你的学校（如果没有自动识别，手动搜一下学校英文名）。
    * Email: 确认选中你的 edu.cn 邮箱。
4.  **位置确认**：
    * 点击绿色按钮 **Share Location** 。
    * **重点**：此时**绝对不要开VPN**！如果浏览器弹窗询问获取位置权限，请点击“允许” 。

## 📸 第五步：上传证明

点击 Continue 后，系统会让你提供证明材料。

<img width="239" height="292" alt="image" src="https://github.com/user-attachments/assets/54bc4f94-e883-4a27-a53b-a5c97633863c" />

1.  **Proof Type (证明类型)**：选择 **"1. Dated school ID"** 。
2.  **拍照上传**：
    * **痛点**：国内学生证通常是中文，审核也就是卡在这里。
    * **解决方案**：我们需要手写或打印一张**英文说明** 。

**请准备一张纸，按以下格式手写或打印：**

> **注意**：
> 1. 请将下方的 `xxx` 替换为你**真实的**信息。
> 2. "Distance Learning" (远程学习) 是为了解决你当前IP可能不在学校宿舍（比如寒暑假在家）的问题。

**模板 1 (用于第一次拍照上传)** ：

```text
Student ID Card
---------------------------
Name: [你的姓名拼音, 如: wang xiaoming]
School: [学校英文全名]
Student ID:
Graduation date:
Study from: Distance Learning
```

## 🔄 第六步：二次验证

上传后，系统大概率会弹出一个补充询问框，问你为什么不在学校）。

<img width="233" height="291" alt="image" src="https://github.com/user-attachments/assets/cea8cf31-7598-4cd1-8e9d-f8188d2d974b" />

1.  **原因选择**：选择 **勾选 "All coursework is via distance learning" (所有课程通过远程进行) 。** 。
2.  **Brief explanation (简短说明)**：填写 **I study via distance learning**。
3.  **再次拍照**：系统可能会要求你再传一次证明。这次我们可以换一个更有针对性的英文说明模板，强调允许远程学习。

**模板 2 (用于补充拍照) ：** ：

```text
Student [你的姓名拼音],
is allowed to study via distance learning in [学校英文全名] University.
```

## ✅ 第七步：等待审核

提交完成后，你应该会看到绿色的 Approved 提示 。

<img width="323" height="236" alt="image" src="https://github.com/user-attachments/assets/cfb7c266-87eb-49dc-9e00-b3a81287cede" />

1. **审核时间**：通常显示 "Approved" 后，权益会在 72小时内 发放到账户 。

2. **有效期**：一般为2年，到期后只要还在读，可以重新认证 。

3. **权益查看**：收到邮件后，访问GitHub查看你领取的 Copilot、JetBrains 全家桶、DigitalOcean 等福利。

## ❓ 常见问题 (Q&A)
**Q: 必须改用户名为真名吗？**

A: 教程中提到了修改 username。如果是新号，建议改为拼音以增加通过率；如果是老号（有很多仓库），改名有风险，建议只改 Display Name (个人资料里的 Name) 即可。

**Q: 拍照模糊怎么办？**

A: 光线要好，字迹要清晰。如果电脑摄像头太糊，可以尝试用手机浏览器登录 GitHub 进行操作。

**Q: 为什么提示我位置太远？**

A: 确保关了梯子。如果还是不行，必须严格按照上述“Distance Learning”的流程操作，那个选项就是专门给不在校区的学生准备的。
